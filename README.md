# BLOG NEXTJS FRONTEND

A full-featured blog platform built with Next.js, offering authentication, content management, and user interaction.

## About the Project

This project is a blog application developed using modern web technologies, focusing on user experience, performance, and real-world application features.

Users can create accounts, log in, publish posts, interact through comments, and manage their own content — simulating a complete blog ecosystem.

## Features
### Authentication System
User login and registration
Secure access to protected routes
### Post Management
Create new blog posts
Edit existing posts
Delete posts
View all publications
Comments System
Users can comment on posts
Interaction between users
### User Profile
Access personal profile
View user-related content
### Responsive Design
Optimized for mobile and desktop
### Technologies Used
Next.js
React
JavaScript / TypeScript
HTML5 & CSS3
Tailwind CSS (if applicable)
Git & GitHub

## Getting Started
### Prerequisites
Node.js
npm or yarn
### Installation

```
# Clone the repository
git clone https://github.com/analuizamelodev/projeto-next.git

# Navigate to the project folder
cd projeto-next

# Install dependencies
npm install

# Run the development server
npm run dev
```

Access the application at:

```
http://localhost:3001
```
